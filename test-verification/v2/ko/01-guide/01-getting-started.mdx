# 검증 테스트 v2에 오신 것을 환영합니다

이것은 검증 테스트 프로젝트의 버전 2.0입니다. 이 버전에서는 향상된 기능과 개선된 사용자 경험을 제공합니다.

## 버전 2.0의 새로운 기능

- **향상된 다국어 지원**: 더 많은 언어 지원
- **개선된 내비게이션**: 더 직관적인 사이드바 구조
- **성능 최적화**: 빠른 로딩 속도
- **모바일 친화적**: 더 나은 모바일 경험

## 시작하기

버전 2.0에서는 다음과 같은 새로운 구조를 사용합니다:

```
src/content/docs/
└── v2/
    ├── en/                # 영어 문서
    ├── ja/                # 일본어 문서
    └── ko/                # 한국어 문서
        ├── 01-guide/      # 가이드
        └── 02-troubleshooting/  # 문제 해결
```

## 주요 개선 사항

### 1. 향상된 검색 기능
- 전체 텍스트 검색
- 다국어 검색 지원

### 2. 개선된 UI/UX
- 다크 모드 지원
- 접근성 개선
- 반응형 디자인 최적화

### 3. 성능 최적화
- 이미지 최적화
- 코드 분할
- 캐싱 개선

## 마이그레이션 가이드

v1에서 v2로 업그레이드하는 경우:

1. 기존 콘텐츠를 새 구조로 이동
2. 설정 파일 업데이트
3. 새로운 기능 활용

자세한 내용은 마이그레이션 가이드를 참조하세요.
