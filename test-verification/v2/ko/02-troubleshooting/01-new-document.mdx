# 일반적인 문제 해결

이 문서는 검증 테스트 v2 사용 중 자주 발생하는 문제와 해결 방법을 설명합니다.

## 빌드 관련 문제

### 빌드 실패

**증상**: `pnpm build` 실행 시 오류 발생

**원인**: 
- 의존성 버전 충돌
- 환경 변수 누락
- TypeScript 타입 오류

**해결 방법**:
1. 의존성 재설치
   ```bash
   rm -rf node_modules
   pnpm install
   ```

2. TypeScript 타입 체크
   ```bash
   pnpm typecheck
   ```

3. 환경 변수 확인
   ```bash
   # .env 파일 존재 여부 확인
   ls -la .env
   ```

### 메모리 부족 오류

**증상**: "JavaScript heap out of memory" 오류

**해결 방법**:
```bash
# Node.js 메모리 제한 증가
NODE_OPTIONS="--max-old-space-size=4096" pnpm build
```

## 개발 서버 문제

### 포트 충돌

**증상**: "Port 4321 is already in use"

**해결 방법**:
1. 다른 포트 사용
   ```bash
   pnpm dev --port 3000
   ```

2. 기존 프로세스 종료
   ```bash
   # 포트 사용 중인 프로세스 찾기
   lsof -i :4321
   
   # 프로세스 종료
   kill -9 <PID>
   ```

### 핫 리로드 작동 안 함

**증상**: 파일 변경 시 자동 새로고침이 되지 않음

**해결 방법**:
1. 개발 서버 재시작
2. 브라우저 캐시 삭제
3. 파일 권한 확인

## 콘텐츠 관련 문제

### 한글 폰트 깨짐

**증상**: 한글이 제대로 표시되지 않음

**해결 방법**:
1. 웹폰트 설정 확인
2. CSS 문자 인코딩 설정
3. 브라우저 인코딩 설정

### 사이드바 메뉴 안 보임

**증상**: 새로 추가한 페이지가 사이드바에 표시되지 않음

**해결 방법**:
1. 파일 이름 규칙 확인 (번호-이름.mdx)
2. frontmatter 설정 확인
3. 사이드바 재생성
   ```bash
   pnpm build:sidebar
   ```

## 성능 문제

### 페이지 로딩 느림

**원인**:
- 큰 이미지 파일
- 많은 JavaScript 번들
- 네트워크 지연

**해결 방법**:
1. 이미지 최적화
2. 코드 분할 적용
3. CDN 사용 고려

## 배포 관련 문제

### Cloudflare Pages 배포 실패

**해결 방법**:
1. 빌드 명령어 확인: `pnpm build`
2. 출력 디렉터리 확인: `dist`
3. 환경 변수 설정 확인

## 추가 도움

문제가 해결되지 않으면:
1. GitHub Issues 확인
2. 개발팀에 문의
3. 디버그 로그 수집
