# Tabs Component Example

This page demonstrates the usage of the Tabs component in documentation.

## Basic Usage

<Tabs>
  <TabItem label="HTML">
    ```html
    <!DOCTYPE html>
    <html>
      <head>
        <title>Hello World</title>
      </head>
      <body>
        <h1>Hello, world!</h1>
      </body>
    </html>
    ```
  </TabItem>
  <TabItem label="CSS">
    ```css
    body {
      font-family: system-ui, sans-serif;
      color: #333;
      background-color: #f9f9f9;
    }
    
    h1 {
      color: navy;
    }
    ```
  </TabItem>
  <TabItem label="JavaScript">
    ```js
    document.addEventListener('DOMContentLoaded', () => {
      console.log('Hello, world!');
    });
    ```
  </TabItem>
</Tabs>

## Synchronized Tabs

These tabs are synchronized across the page using the `syncKey` prop.

<Tabs syncKey="languages">
  <TabItem label="JavaScript">
    ```js
    // JavaScript example
    function greet(name) {
      return `Hello, ${name}!`;
    }
    
    console.log(greet('World'));
    ```
  </TabItem>
  <TabItem label="TypeScript">
    ```ts
    // TypeScript example
    function greet(name: string): string {
      return `Hello, ${name}!`;
    }
    
    console.log(greet('World'));
    ```
  </TabItem>
  <TabItem label="Python">
    ```python
    # Python example
    def greet(name):
        return f"Hello, {name}!"
    
    print(greet("World"))
    ```
  </TabItem>
</Tabs>

Another set of tabs with the same `syncKey` will stay in sync with the above tabs:

<Tabs syncKey="languages">
  <TabItem label="JavaScript">
    JavaScript is a scripting language that enables dynamic content on web pages.
  </TabItem>
  <TabItem label="TypeScript">
    TypeScript is a strongly typed programming language that builds on JavaScript.
  </TabItem>
  <TabItem label="Python">
    Python is a high-level, general-purpose programming language.
  </TabItem>
</Tabs>
